Objetos

Un objeto es un conjunto de propiedades, y una propiedad es una asociación clave-valor. Cabe destacar que en Javascript el valor de una propiedad puede ser una función a la que se le llama método.

Visión general

Los objetos en JavaScript, como en tantos otros lenguajes de programación, se pueden comparar con objetos de la vida real. El concepto de Objetos en JavaScript se puede entender con objetos tangibles de la vida real.

Objetos y propiedades

Un objeto de JavaScript tiene propiedades asociadas a él. Una propiedad de un objeto se puede explicar como una variable adjunta al objeto. Las propiedades de un objeto básicamente son lo mismo que las variables comunes de JavaScript, excepto por el nexo con el objeto ya que estas definen las características del objeto. 


                    objectName.propertyName

Como todas las variables de JavaScript, tanto el nombre del objeto (que puede ser una variable normal) como el nombre de la propiedad son sensibles a mayúsculas y minúsculas. Puedes definir propiedades asignándoles un valor. Por ejemplo, vamos a crear un objeto llamado auto y le vamos a asignar propiedades denominadas marca, modelo, y año de la siguiente manera: 


                    var auto = new Object();
                    auto.marca = 'Ford';
                    auto.modelo = 'Mustang';
                    auto.año = 1969;

El ejemplo anterior también se podría escribir usando un iniciador de objeto, que es una lista delimitada por comas de cero o más pares de nombres de propiedad y valores asociados de un objeto, encerrados entre llaves ({}): 


                    var auto = {
                        marca: 'Ford',
                        modelo: 'Mustang',
                        año: 1969
                    };

Las propiedades no asignadas de un objeto son undefined (y​no null). 


                    auto.color; // undefined

También se puede acceder o establecer las propiedades de los objetos en JavaScript mediante la notación de corchetes ↑[]↓. Los objetos, a veces son llamados arreglos asociativos, debido a que cada propiedad está asociada con un valor de cadena que se puede utilizar para acceder a ella. Por lo tanto, por ejemplo, se puede acceder a las propiedades del objeto auto de la siguiente manera: 


                    auto['marca']  = 'Ford';
                    auto['modelo'] = 'Mustang';
                    auto['año']  = 1969;

También se puede acceder a las propiedades mediante el uso de un valor de cadena que se almacena en una variable:

En el siguiente ejemplo podemos ver que se llama a un metodo dos veces, primero cuando se carga la pagina y otra cuando el usuario hace click en un botón. 


                var propertyName = 'marca';
                auto[propertyName] = 'Ford';

                propertyName = 'modelo';
                auto[propertyName] = 'Mustang';
