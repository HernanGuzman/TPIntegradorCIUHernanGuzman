Variables

Una variable es un elemento que se emplea para almacenar y hacer referencia a otro valor, mas en concreto es un espacio de memoria que se reservada para almacenar información necesaria para la ejecución de un programa. Gracias a las variables es posible crear "programas genéricos", es decir, programas que funcionan siempre igual independientemente de los valores concretos utilizados.

De la misma forma que si en Matemáticas no existieran las variables no se podrían definir las ecuaciones y fórmulas, en programación no se podrían hacer programas realmente útiles sin las variables.

Ejemplo de programa sin la utilización de variables: 


                    resultado = 3 + 1

Este calculo solo se podria utilizar para realizar la operación de 3 + 1. Si desearamos realizar otro calculo deberiamos cambiar el codigo.

Ahora el ejemplo utilizando varables: 


                    numero_1 = 3
                    numero_2 = 1
                    resultado = numero_1 + numero_2

Ahora el calculo se realiza en función del valor que contienen las varables numero_1 y numero_2. Este programa funcionará correctamente para cualquier valor que se almacene en las variables. Los elementos numero_1 y numero_2 son variables que almacenan los valores que utiliza el programa.

Las variables en JavaScript se crean mediante la palabra reservada var. De esta forma, el ejemplo anterior se puede realizar en JavaScript de la siguiente manera: 


                    var numero_1 = 3;
                    var numero_2 = 1;
                    var resultado = numero_1 + numero_2;

La palabra reservada var solamente se debe indicar al definir por primera vez la variable, lo que se denomina declarar una variable. Cuando se utilizan las variables en el resto de instrucciones del script, solamente es necesario indicar su nombre. En otras palabras, en el ejemplo anterior sería un error indicar lo siguiente: 


                    var numero_1 = 3;
                    var numero_2 = 1;
                    var resultado = var numero_1 + var numero_2;

El nombre de una variable también se conoce como identificador y debe cumplir las siguientes normas:

Sólo puede estar formado por letras, números y los símbolos $ (dólar) y _ (guión bajo). El primer carácter no puede ser un número. Por tanto, las siguientes variables tienen nombres correctos: 


                    var $numero1;
                    var _$letra;
                    var $$$otroNumero;
                    var $_a__$4;
