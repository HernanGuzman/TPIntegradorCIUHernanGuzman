Arrow function (Función flecha)

Las funciones de flecha se introdujeron en ES6 y nos permiten escribir una sintaxis de función más corta:

Función normal


                    hello = function() {
                        return "Hello World!";
                      }

Función flecha


                    hello = () => {
                        return "Hello World!";
                      }

Esta función se puede declarar mas corta si tiene una sola declaración y esta declaración devuelve un valor de la siguiente manera:

Función flecha


                    hello = () => "Hello World!";

Si la función recibe parametros se puede declarar de dos formas 


                hello = (val) => "Hello " + val;

                //o sin paréntesis
                
                hello = val => "Hello " + val;

¿Qué pasa con this?

El manejo de this en las funciones de flecha también es diferente en comparación con las funciones normales. En las funciones flechas no hay vinculación de this. En el caso de las funciónes normales this hace referencia al objeto que llama a la función, que puede ser un botón, un documento, etc.

En el caso de las funciones flecha la palabra this hace referencia al objeto que definio la función flecha.

En el siguiente ejemplo podemos ver que se llama a un metodo dos veces, primero cuando se carga la pagina y otra cuando el usuario hace click en un botón.

Con función regular:

Devolvera dos objetos diferentes (ventana y botón) 


            // Regular Function:
            hello = function() {
                document.getElementById("demo").innerHTML += this;
            }

            // The window object calls the function:
            window.addEventListener("load", hello);

            // A button object calls the function:
            document.getElementById("btn").addEventListener("click", hello);

Con función flecha:

Devolvera dos veces el objeto ventana ya que es el propietario de la función. 

    
                // Arrow Function:
                hello = () => {
                    document.getElementById("demo").innerHTML += this;
                }
    
                // The window object calls the function:
                window.addEventListener("load", hello);
    
                // A button object calls the function:
                document.getElementById("btn").addEventListener("click", hello);
