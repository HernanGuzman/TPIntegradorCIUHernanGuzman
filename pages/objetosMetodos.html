Metodos de objetos

Un método es una función asociada a un objeto, o, simplemente, un método es una propiedad de un objeto que es una función. Los métodos se definen normalmente como una función, con excepción de que tienen que ser asignados como la propiedad de un objeto. Un ejemplo puede ser: 


                objectName.methodname = functionName;

                var Persona = {
                  Caminar: function(params) {
                    // ...hacer algo
                  }
              
                  // O ESTO TAMBIÉN FUNCIONA
              
                  Saltar(params) {
                    // ...hacer algo más
                    }
                };

donde Persona es un objeto existente, Caminar es el nombre que se le va a asignar al método.

Entonces se puede llamar al método en el contexto del objeto de la siguiente manera: 


                object.methodname(params);

Se puede definir métodos para un tipo de objeto incluyendo una definición del método en el constructor del objeto. Se podria crear un metodo que muestre los datos de las personas previamente definidas; por ejemplo: 


                    function mostrarPersona() {
                        var result = `Nombre: ${this.nombre}, direccion ${this.make}, teléfono ${this.model}`;
                        pretty_print(result);
                      }

pretty_print es una función de Javascript para mostrar una línea horizontal y una cadena.

Se puede hacer de esta función un método de Persona agregando la declaración 


                    this.mostrarPersona = mostrarPersona;

a la definición del objeto. Por lo tanto, la definición completa de Persona ahora se verá así: 


                    function Persona(nombre, direccion, telefono) {
                        this.nombre = nombre;
                        this.direccion = direccion;
                        this.telefono = telefono;
                        this.mostrarPersona = mostrarPersona;
                      }

Entonces se puede llamar al método mostrarPersona para cada uno de los objetos de la siguiente manera: 


                    Persona1.mostrarPersona();
                    Persona2.mostrarPersona();
